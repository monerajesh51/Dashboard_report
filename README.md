# Dashboard_report on IPL-2025 Dataset using PowerBI
IPL 2025 Dashboard Report using Power BI
Project Overview

This project presents an interactive Power BI dashboard built using IPL 2025 dataset to analyze team performance, player statistics, match outcomes, and overall tournament trends. The dashboard provides a comprehensive view of the season through dynamic visualizations and key performance indicators (KPIs), enabling users to gain meaningful insights into team and player performances.

Objectives
Analyze team and player performance throughout the IPL 2025 season.
Track key tournament statistics and match outcomes.
Identify top-performing batsmen and bowlers.
Monitor scoring patterns and average run rates.
Provide actionable insights through interactive dashboards.
Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Power Query
Microsoft Excel (Data Preparation)
Key Features
Interactive filters and slicers for team-wise and match-wise analysis.
Top Run Scorers analysis.
Leading Wicket Takers analysis.
Team Performance comparison.
Match Results and Win Statistics.
Average Run Rate analysis.
KPI Cards for important tournament metrics.
Dynamic visualizations for enhanced data exploration.
Data Processing
Cleaned and transformed raw IPL 2025 data using Power Query.
Handled missing values and standardized data formats.
Created calculated columns and DAX measures for advanced analytics.
Designed relationships between tables for accurate reporting.
Dashboard Insights
Identified the highest run-scoring players of the season.
Analyzed bowling performance and wicket-taking trends.
Compared team performances based on wins, losses, and points.
Evaluated match scoring patterns and average run rates.
Highlighted key tournament trends and standout performances.
Outcome

The dashboard provides a clear and data-driven overview of the IPL 2025 season, helping users understand player contributions, team performance, and match dynamics through intuitive visualizations and interactive reporting.
